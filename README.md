# 🎭 Playwright Automation Portfolio

## 📌 Overview

This project demonstrates my skills in **UI test automation using Playwright**.
It includes end-to-end tests, UI validations, and reusable test structures built with real-world scenarios.

The goal of this repository is to showcase:

* Test design and structure
* Automation best practices
* Cross-browser testing
* CI-ready setup

---

## ⚙️ Tech Stack

* Playwright
* JavaScript / TypeScript
* Node.js
* HTML Reporter

---

## 📂 Project Structure

```
tests/
  ui/                # UI component tests (forms, tables, dynamic elements)
  user-flows/        # End-to-end scenarios
  validations/       # Input validations and edge cases
  demoqa/            # Practice site tests
```

---

## 🚀 Getting Started

### Install dependencies

```
npm install
```

### Run tests

```
npx playwright test
```

### Run in headed mode

```
npx playwright test --headed
```

### Open HTML report

```
npx playwright show-report
```
